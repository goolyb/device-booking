# device-booking


## Project scope

- Develop a web-based application for managing a list of MacBooks.
- Allow users to add and remove MacBooks from the inventory list.
- Provide a web interface where the lists can be viewed and edited.

## Technical Requirements

1. The system must allow users to add MacBooks to the inventory list.
2. The system must allow users to delete MacBooks from the inventory list.
3. The system must allow an administrator to edit MacBook entries in the inventory list.
4. The system must display the inventory list with device ID, username, status, and pickup time.
5. The system must require user authentication to access the website.
6. The system must store all MacBook inventory records in a database.
7. The system must validate input data before adding a new MacBook entry to the inventory.

## Non-Technical Requirements

1. The interface should be simple and intuitive for all users.
2. Editing and deleting entries should require minimal steps for the user.
3. All information should be clean and easy to read for every user.
4. The inventory list should be well-organized so that all users can navigate and find information easily.
5. The website should be accessible through standard web browsers without installing additional software.
6. The system should provide basic feedback when an action is completed or fails.
